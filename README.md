## Getting started

Download example data from [Kaggle Stack Sample](https://www.kaggle.com/stackoverflow/stacksample) into the folder `data/`

Then setup your dev environment:

```bash
python3 tasks.py init
. ./.venv/bin/activate
invoke test
```
